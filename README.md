# Security Log Analyzer 🛡️

أداة برمجية مبنية بلغة Python لتحليل سجلات الدخول (Logs) واكتشاف محاولات الدخول غير المصرح بها.

### 🔍 الميزات الرئيسية:
* **تصفية البيانات:** استخراج محاولات الدخول الفاشلة بدقة.
* **التحليل الإحصائي:** حساب تكرار المحاولات الفاشلة لكل عنوان IP لتحديد الأنشطة المشبوهة.
* **سهولة الاستخدام:** كود نظيف وقابل للتطوير.

### 🛠 التقنيات المستخدمة:
* **Python**
* **Pandas Library** (لتحليل ومعالجة البيانات)

---
*هذا المشروع هو جزء من معرض أعمالي المهنية في مجال تحليل البيانات والأمن السيبراني.*
### 🚀 خريطة الطريق (Future Improvements)
أخطط لتطوير هذه الأداة مستقبلاً لتشمل:
* **التنبيهات التلقائية:** إرسال تنبيه في حال تجاوزت المحاولات الفاشلة عدداً معيناً.
* **الرسوم البيانية:** إضافة مكتبة مثل Matplotlib لتحويل التحليل إلى رسوم بيانية تفاعلية.
* **دعم ملفات أكثر:** توسيع الأداة لتتمكن من قراءة أنواع أخرى من سجلات البيانات (مثل JSON).
