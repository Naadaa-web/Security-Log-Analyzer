# Security Log Analyzer 🛡️

أداة برمجية مبنية بلغة Python لتحليل سجلات الدخول (Logs) واكتشاف محاولات الدخول غير المصرح بها.

### 🔍 الميزات الرئيسية:
* **تصفية البيانات:** استخراج محاولات الدخول الفاشلة بدقة.
* **التحليل الإحصائي:** حساب تكرار المحاولات الفاشلة لكل عنوان IP لتحديد الأنشطة المشبوهة.
* **سهولة الاستخدام:** كود نظيف وقابل للتطوير.

### 🛠 التقنيات المستخدمة:
* **Python**
* **Pandas Library** (لتحليل ومعالجة البيانات)
